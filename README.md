# AI-Powered Personalized Learning Path Generator
## Description
This project is an AI-based system that creates a personalized learning path for users based on their current skills and career goals.
## Problem
Many learners do not know what skills to learn next to reach their career goals.
## Solution
The system analyzes a user’s skills and compares them with the skills required for a chosen career. It then generates a step-by-step learning roadmap.
## Features
- User skill input
- Skill gap analysis
- Personalized learning path
- Course recommendations
## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Machine Learning
## How It Works
1. User enters their current skills.
2. User selects a career goal.
3. The system identifies missing skills.
4. AI generates a learning path.
## Example
Input:
Skills: Python Basics  
Goal: Data Scientist  
Output:
1. Statistics
2. Machine Learning
3. Data Visualization
4. Deep Learning
